## Truffle Tutorial

#### 1️⃣ Step 1

Download and run [Ganache](https://www.trufflesuite.com/ganache).

#### 2️⃣ Step 2

Run these commands.

```
$ npm i truffle -g
$ npm i
$ truffle migrate
$ truffle test
```
